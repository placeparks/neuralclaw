NeuralClaw Session Helper

Purpose:
- Copy your Claude sessionKey or ChatGPT session cookie into the clipboard.
- This extension does not send credentials anywhere. It only reads the browser cookie locally and copies it.

Install:
1. Unzip this folder somewhere on your computer.
2. Open Chrome and go to chrome://extensions
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select the unzipped folder.

Use:
1. Stay logged in to claude.ai or chatgpt.com in this same browser profile.
2. Click the extension icon.
3. Click the matching Copy button.
4. Paste into the NeuralClaw onboarding form.
